JSThon - Python in JS

Run Python-like code directly in the browser with JS libraries

Features

  • Output Console: Displays results and error messages from your code.
  • Brython Integration: Python code runs in the browser via the Brython runtime.
  • JavaScript Access: Python code can access JS globals and libraries using from browser import window.
  • Dynamic Execution: Run Python strings on demand with the jsthon() function.

How to Use JSThon

  1. Run it via <script type="module" src="eotter-beep.github.io/jsthon/jsthon.js"></script>
  2. To run a function in the script go inside of the script <script> ... </script>
